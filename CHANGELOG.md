## [1.0.0] - 2026-01-05

### Added
- Initial release of CyberPanel MCP Server
- 200+ tools for comprehensive server management
- Website management (create, delete, suspend, unsuspend)
- Database management (MySQL/MariaDB)
- Email management (accounts, forwardings, DKIM)
- DNS management (zones, records)
- SSL certificate management (Let's Encrypt, custom)
- File management (browse, edit, upload, download)
- FTP account management
- Firewall management (rules, policies)
- Backup management (local, S3, DigitalOcean Spaces, MinIO)
- Docker container management
- User and package management
- PHP version and configuration management
- WordPress management tools
- Server status and monitoring
- SSH key management
- N8N workflow automation tools
- API Keys plugin for CyberPanel
- One-click configuration for Claude Desktop and VS Code
- Comprehensive documentation and README

### Security
- API key-based authentication
- HTTPS communication
- Secure key storage in CyberPanel database
